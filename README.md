# Custom ESP32 Audio Processing Shield
## 1.) Project Overview
### 1.1) Description
This project aims to create a Audio Processing Shield with an ESP32-S3. It shall implement immediate hardware prototyping, core firmware development, and initial processor subsystem
prototyping phase, focusing on mathematical modeling, 4-layer PCB layout, and bare-metal validation. This shall act as a multi-disciplinary project, tying in PCB Design, Embedded Programming and Digital Signal Processing.
